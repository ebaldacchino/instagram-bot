# Instagram Bot

Hey guys. This is still just a work in progress. In a nutshell, it's a barebones Node JS instagram bot that uses Puppeteer as a headless browser. So far, to work it, you need two environment variables

    IG_USERNAME (your instagram username)
    IG_PASSWORD (your instagram password)

To iterate through instagram tags, you'll need to add them to the array that functions as the *likeTagsProcess* function's argument in index.js

I haven't spent a lot of time on this yet, so if you've got any ideas, feel free to get in touch, send a PR through, or whatever else floats your boat!
